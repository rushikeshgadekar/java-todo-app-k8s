Web Interface

Use the web interface to manage todos with user-friendly forms:

  • Create Todo Form - Add new todos with validation
  • Show All Todos - View all existing todos

API Endpoints

You can also interact with the API directly:

POST /todo/create
Create a new todo (JSON)
GET /todo/todos
Get all todos
GET /todo/{id}
Get todo by ID
PUT /todo/update
Update existing todo
DELETE /todo/delete/{id}
Delete todo by ID
